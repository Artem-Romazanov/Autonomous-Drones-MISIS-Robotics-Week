# Выполнение хакатона «Автономные дроны и бортовой ИИ» | MISIS Robotics Week
В данном репозитории представлены все файлы решения команды misis_team для хакатон-интенсива «Автономные дроны и бортовой ИИ» от компании «Сверх» в рамках мероприятия MISIS Robotics Week!



# Команда проекта:
[Голышев Юрий Александрович ](https://github.com/wodocanal)— отвечал за интеграцию ROS 2 с полетным контроллером, адаптацию алгоритма SLAM для построения карты лабиринта и реализацию логики автономного полета с аварийной защитой.

[Симакова Анастасия Александровна ](https://github.com/fismay)— руководила разработкой, отвечала за интеграцию YOLO для распознавания объектов, настройку пайплайна компьютерного зрения и калибровку камеры, координировала работу команды.

[Ромазанов Артём Валентинович](https://github.com/Artem-Romazanov)— обеспечивал техническую подготовку: настройку Ubuntu и ROS 2, тестирование сенсоров, помогал с настройкой окружения для ML и готовил демонстрационные материалы.

[Тамбовцев Савва Викторович](https://github.com/CardlnaI)— занимался дообучением YOLO, тестировал качество распознавания,настройку пульта с Kill-switch, ассистировал при полетных тестах и анализировал логи.
